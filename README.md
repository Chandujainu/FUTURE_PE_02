# AI Content Marketing using UGC Ads

## Future Interns – Prompt Engineering Internship

### Task 2: AI Content Marketing using UGC Ads

## Project Overview
This project demonstrates the use of AI-powered prompt engineering to create engaging User-Generated Content (UGC) advertisement scripts for a local business. The content is designed to help businesses promote their services through authentic and persuasive social media marketing.

## Business Name
Radiance Beauty Salon

## AI Tool Used
- ChatGPT

## Objective
To generate high-quality UGC advertisement scripts that can be used on Instagram, Facebook, YouTube Shorts, and TikTok to increase customer engagement and brand awareness.

## Deliverables
- 5 UGC Advertisement Scripts
- Hook
- Customer Problem
- Personal Experience
- Solution
- Benefits
- Call-to-Action (CTA)
- Social Media Caption
- Relevant Hashtags

## Services Promoted
- Hair Spa
- Bridal Makeup
- Facial Treatment
- Hair Styling
- Nail Art

## Files Included
- UGC_Ads.pdf
- Prompts.txt
- README.md

## Skills Learned
- Prompt Engineering
- AI Content Generation
- Copywriting
- Social Media Marketing
- Digital Marketing

## Conclusion
This project demonstrates how AI can be used to generate engaging marketing content quickly and efficiently. The generated UGC advertisements are suitable for promoting beauty salon services across multiple social media platforms.
